# Ahsan Auto - Car Dealer Website

## Overview
Welcome to Ahsan Auto, your premium destination for luxury cars. Our website provides a seamless experience for browsing, searching, and purchasing high-end vehicles.

## Features
- **Browse Cars**: Explore a wide range of luxury cars with detailed specifications.
- **Advanced Search**: Filter cars by make, model, year, price, and more.
- **Car Details**: View comprehensive details and high-quality images of each car.
- **User Authentication**: Secure login and registration for personalized services.
- **Admin Dashboard**: Manage car listings, user inquiries, and website settings.

## Technologies Used
- **Frontend**: React, Next.js, Tailwind CSS, Framer Motion
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JWT Tokens
- **API**: RESTful API with JSON data exchange

## Getting Started

### Prerequisites
- Node.js
- npm or Yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/ahsan-auto.git
   ```
2. Install dependencies:
   ```bash
   cd ahsan-auto
   npm install
   # or
   yarn install
   ```

### Running the Application
- Development Mode:
  ```bash
  npm run dev
  # or
  yarn dev
  ```
- Production Mode:
  ```bash
  npm run build
  npm start
  # or
  yarn build
  yarn start
  ```
